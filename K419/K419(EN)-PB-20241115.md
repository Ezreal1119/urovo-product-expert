# K419 — Mobile Printer

## Product Highlights

**Bluetooth 4.2** · **90mm/s Print Speed** · **1600mAh Battery** · **56mm Media Roll** · **Multi-Media Support**

## Key Features

### Stable Connectivity
Bluetooth 4.2 enables stable connections between multiple devices for flexible mobile printing.

### Versatile Printing
Supports multiple languages, numbers, graphics, and barcodes. Print speed up to 90 mm/s.

### Wide Media Compatibility
Handles continuous, die-cut, notch, and black mark media. Easily prints labels, tags, and inlays.

### All-Day Power
1600 mAh high-capacity battery supports full-shift workloads.
