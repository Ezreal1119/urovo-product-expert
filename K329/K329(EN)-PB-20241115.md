# K329 — Mobile Printer

**120mm/s Print Speed** · **Bluetooth 4.2 LE + NFC** · **2-3 inch** · **Receipts & Labels** · **Portable**

## Key Features

### Newly Designed, Class-Leading
Portable 2–3 inch mobile printer with 120 mm/s print speed. Class-leading performance in a compact form factor.

Supports both receipt and label printing for versatile deployment across retail, transportation, and manufacturing.

### Quick Pairing
Bluetooth 4.2 Low Energy and passive NFC tags for instant pairing with mobile devices.

Long-life battery with intelligent power consumption control for extended work shifts.

### Multi-Language Compatible
Compatible with multiple programming languages, reducing re-development effort. Visual editing apps for design-at-view, easy printing.
