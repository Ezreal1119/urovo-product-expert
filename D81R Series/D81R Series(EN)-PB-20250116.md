# D81R Series — Desktop UHF RFID Printer

## Product Highlights

**UHF RFID Read/Write** · **203/300 DPI** · **Dual Print Mode** · **Simultaneous Print & Encode** · **USB + Ethernet**

## Key Features

### Integrated UHF RFID
Supports simultaneous printing and encoding of RFID labels — no need to rewind labels, simplifying the printing process.

### Dual Printing Mode
Direct thermal and thermal transfer modes for seamless adaptation to different media needs.

### Compact Label Support
Print and encode RFID labels with minimum spacing, enabling efficient high-density label production.

### Versatile Connectivity
USB + USB Host + Ethernet + Serial standard. Wi-Fi and Bluetooth optional.
