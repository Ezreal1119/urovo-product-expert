# Impinj E710 — UHF RFID Chipset

The Impinj E710 is the flagship UHF RFID reader chip used across UROVO's high-performance RFID product line. It delivers industry-leading read rates and range.

## Specifications

| Parameter | Value |
| --- | --- |
| Read Rate | 1,300+ tags/second |
| Gen2X Support | Yes (on newer implementations) |
| Frequency Range | 840–960 MHz (region-adjustable) |
| Protocol | ISO18000-6C (EPC Class1 Gen2) |
| Max Output Power | Up to 33 dBm |
