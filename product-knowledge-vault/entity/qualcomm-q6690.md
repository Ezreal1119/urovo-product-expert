# Qualcomm Q-6690 — Enterprise Mobile Processor

The Qualcomm Q-6690 is a next-generation enterprise mobile platform with fully integrated UHF RFID (RAIN) capabilities — the world's first enterprise processor with this level of RFID integration. Powers UROVO's flagship DT610 series.

| Parameter | Value |

| AI Performance | 6 INT8 TOPS |

## Key Capabilities

- **Edge AI**: 6 TOPS for real-time OCR, speech, computer vision
- **Integrated UHF RFID**: Impinj Gen2X, 50+ tags/sec, 1.5 m range (no external chip needed)
- **Connectivity**: 5G NR Sub-6 (NSA+SA), Wi-Fi 7 (802.11be), Bluetooth 6.0
- **Memory**: Up to 12 GB RAM + 256 GB UFS storage
