# S710 — Barcode Scanner

> Source: S710(EN)-SPEC-20230411.md

Category: [[categories/barcode-scanner|Barcode Scanner]]


## Performance

| Parameter | Specification |
| --- | --- |
| Operation System | Compatible with Windows, Android, Linux |
| Processor | 32-bit ARM |
| Camera | 640 × 480 CMOS |
| Viewing Angle | Horizontal 31° |
| Scan Angles | Roll 360°, Skew ±55°, Pitch ±55° |

## Decode Capabilities

| Type | Supported |
| --- | --- |
| 1D/2D | EAN, QR, GS1 and mainstream barcodes |
| Screen Codes | Barcodes on monitors and mobile devices |
| Payment Codes | All payment codes, electronic tags, coupons, color barcodes |

## Reading Distance

| Barcode | Range |
| --- | --- |
| EAN-13 (13 mil) | 4.0–20.0 cm |
| Code 128 (15 mil) | 4.5–25.0 cm |
| Code 39 (5 mil) | 4.5–9.5 cm |
| QR Code (15 mil) | 4.0–18.0 cm |

## Physical & Power

| Parameter | Specification |
| --- | --- |
| Weight | 185 g (including cable) |
| Cable Length | 1.8 m |
| Operating Voltage | USB 5V |
| Operating Current | 125 mA |
| Standby Current | 25 mA |
