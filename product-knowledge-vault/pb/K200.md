# K200 — Industrial Wired Barcode Scanner

> **Category**: [[categories/barcode-scanner|Barcode Scanner]] | **Specs**: [[spec/K200|K200 Specification]]

## Product Highlights

**Professional Scan Engine** · **DPM Code Reading** · **Dual-Color Illumination** · **Plug & Play USB**

## Key Features

### Powerful Decoding
Professional scanning engine easily acquires DPM codes on all surfaces — laser engraving, dot peening, chemical etching, and ink marking — plus all standard 1D/2D barcodes.

### Dual-Color Variable Illumination
Switchable red and white LED system automatically adjusts based on barcode type and surface.

### Plug to Use
USB connectivity with multi-system compatibility — simply connect and start scanning.
