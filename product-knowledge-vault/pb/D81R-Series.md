# D81R Series — Desktop UHF RFID Printer

> **Category**: [[categories/desktop-printer|Desktop Printer]] | **Specs**: [[spec/D81R-Series|D81R Series Specification]]

## Product Highlights

**UHF RFID Read/Write** · **203/300 DPI** · **Dual Print Mode** · **Simultaneous Print & Encode**

## Key Features

### Integrated UHF RFID
Simultaneous printing and encoding of RFID labels — no rewind needed.

### Dual Printing Mode
Direct thermal and thermal transfer modes for different media needs.

### RFID-Specific Hardware
RFID print/write speed, memory, CPU, and media handling differ from D8100 Plus; use the D81R spec page for RFID tender responses.
