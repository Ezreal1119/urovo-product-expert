# SR5600 V2 — Wearable Ring Scanner Specification

## Scanner

| Parameter | Specification |
| --- | --- |
| Image Sensor | 1280 × 1080 CMOS, 60 fps |
| Illumination | White LED |
| Aimer | Red Laser, 650 nm |
| Field of View | 38°(H) × 33°(V) |
| Scanning Angle | Roll 360°, Skew ±65°, Pitch ±65° |
