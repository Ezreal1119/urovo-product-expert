# DT40 — Enterprise Mobile Computer

## Product Highlights

**IP67 Sealed** · **4500mAh Battery** · **4" Touch Screen + Keypad** · **Fast Wi-Fi Roaming** · **Advanced Scan Engine**

## Key Features

### 4" Touch Screen & Keypad Dual Input
Combines a 4-inch touch display with a full 21-key numeric keypad — offering flexible dual input for fast data entry. Compact and chic design for all-day comfortable use.

### Fast & Seamless Wi-Fi Roaming
Switches between multiple access points rapidly and stably, ensuring uninterrupted connectivity for mobile workflows.

### Bold Sound Feedback
Enhanced high-power speaker delivers clear audio feedback even in noisy environments. Supports audio, vibration, and LED notifications.

### Excellent Performance
Powered by an octa-core processor running Android 12, delivering smooth application performance for enterprise tasks.

### Precise 1D/2D Decoding
Advanced scan engine with an extended scanning range from 5.6 cm to over 12.2 m, supporting all international standard 1D/2D barcodes.
