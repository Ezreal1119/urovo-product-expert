# FR2000 — Desktop RFID Reader

## Product Highlights

**Impinj E710** · **99.9% Read Accuracy** · **Anti Cross-Read** · **USB + PoE Power** · **Full-Frequency UHF**

## Key Features

### Exceptional Performance
Equipped with Impinj E710 chip and proprietary algorithms, achieving up to 99.9% read accuracy for reliable RFID operations.

### Anti Cross-Read Design
Robust metal casing prevents signal leakage, significantly improving reading accuracy while eliminating cross-reads in multi-tag environments.

### Versatile Power Options
USB Type-C power supply with optional PoE support. Single cable for both data and power via USB-C or RJ45 Ethernet — reducing deployment and maintenance costs.

### Full-Frequency UHF Tag Read & Write
Supports multiple operating frequencies (840–960 MHz) and ISO 18000-6C (EPC C1G2) international protocol for full-frequency UHF tag compatibility.
